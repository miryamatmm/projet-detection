# Détection d’Anomalies : Fraudes Bancaires & Intrusions Réseau

Projet de Data Science visant à détecter des comportements anormaux dans deux contextes réels :
- transactions bancaires frauduleuses (Credit Card Fraud),
- intrusions réseau (KDDCup99).

Ce travail combine des approches **non supervisées** (IsolationForest, LOF) et **supervisées** (Régression Logistique, XGBoost, Random Forest), avec une attention particulière à la **gestion du déséquilibre des classes** et à l’évaluation des performances.
Le rapport détaillé contenant toutes les étapes, analyses, visualisations et résultats est disponible dans le dossier du projet.
